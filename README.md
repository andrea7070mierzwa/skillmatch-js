# SkillMatch JS 😄

Mini projeto avaliativo desenvolvido para o curso
**Desenvolvedor Frontend React - SENAI / SCTEC**

---

## Sobre o projeto

O SkillMatch JS é um sistema criado para realizar o cálculo de compatibilidade entre candidatos e vagas de Frontend Developer Junior.

A proposta do sistema é comparar:

* HardSkills
* SoftSkills
* Percentual de compatibilidade
* Regras de aprovação
* Recomendações para desenvolvimento profissional

Tudo isso utilizando JavaScript puro, lógica de programação e Programação Orientada a Objetos.

---

## Funcionalidades

✅ Cadastro de vagas
✅ Cadastro de candidatos
✅ Comparação de HardSkills e SoftSkills
✅ Cálculo percentual de compatibilidade
✅ Classificação de candidatos
✅ Feedback personalizado
✅ Regras de aprovação
✅ Simulação assíncrona com Promise e async/await

## Como executar o projeto

Clone o repositório:

```bash
git clone LINK_DO_REPOSITORIO
```

Acesse a pasta do projeto:

```bash
cd skillmatch-js
```

Execute o sistema:

```bash
node skillmatch.js
```

## Diário de bordo do caos organizado 😂

Este projeto começou simples.

Depois teve:

* herança desnecessária,
* arrays redundantes,
* classes que nasceram e morreram no mesmo dia,
* variáveis em lugares duvidosos,
* um pequeno surto coletivo com o terminal Linux,
* e alguns momentos de:

> “daqui a pouco não tenho mais código”

😅

No processo, descobri que:

* menos código às vezes é melhor;
* arquitetura bonita demais pode virar decoração;
* GitHub gosta de brincar de esconde-esconde;
* e o terminal definitivamente não foi feito pensando em seres humanos emocionalmente estáveis.
